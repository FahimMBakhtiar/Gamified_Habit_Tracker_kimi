# Gamified_Habit_Tracker_kimi
Your LifeQuest RPG — a cinematic, gamified Life Habit Tracker inspired by the whimsical, handcrafted fantasy world of It Takes Two

**What Was Built**
Immersive Fantasy RPG Shell
Deep Enchanted Purple (#3D2A4E) base with Amber Gold (#D97706) accents and Moss Green (#7A8B69) highlights — a warm, cinematic color palette
Custom Cinzel serif typography for headlines, Quattrocento Sans for body — storybook-inspired
Ambient floating particles, wood-panel textures, parchment cards with 3D shadows
Framer Motion screen transitions (spring physics), glowing stat vials, pulsing campfire effects
Chunky tactile buttons with squash-and-press animations

**6 Fully Interactive Game Screens**

Table
| Screen                  | Purpose             | Key Features                                                                                                                                 |
| ----------------------- | ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| **Quest Board**         | Habit tracking      | 8 pre-built quests across 7 categories, swipe-to-complete with XP burst particles, streak counters, filter tabs, animated progress ring      |
| **Character Dashboard** | Stats & progression | 10 RPG stats (Strength, Wisdom, Fitness, etc.) with liquid-filled vials, 8 skill nodes with upgrade system, 8 achievements with rarity tiers |
| **Treasury**            | Finance tracker     | Net worth visualization, income/expense tracking, 5 budget categories with animated bars, wealth growth chart, financial milestones          |
| **Guild Hall**          | Social/community    | 4 guild members with leaderboard, class icons, streak & contribution stats, 3 guild challenges with shared progress                          |
| **World Map**           | World evolution     | Evolution timeline (5 stages from campsite to magical kingdom), 6 biome features with individual growth meters, tips panel                   |
| **Dungeon Battle**      | Focus timer         | 4 preset durations (5/15/25/45 min), animated SVG timer ring, monster HP indicator, victory celebration with loot drops                      |

**Core Game Systems**
1. RPG Progression: Level 12 character with XP bar, gold currency, streak counter, combo bonuses
2. Quest Categories: Spiritual, Fitness, Wisdom, Financial, Social, Discipline, Focus — each with unique XP rewards
3. Difficulty Tiers: Easy/Medium/Hard/Legendary with multipliers
4. Skill Tree: 8 upgradeable skills across all categories, costing gold
5. Achievements: Common through Legendary rarity with unlock tracking
6. World Evolution: 0-100% progression system with 5 visual stages


**7 AI-Generated Fantasy Assets**
Diorama hero village, chibi character avatar, treasure vault, guild hall interior, quest board, evolved world scene, skill tree background, dungeon battle scene

**Tech Stack**
React + TypeScript + Vite + Tailwind CSS + Framer Motion + Lucide React icons

The experience blurs the line between playing a magical adventure game and improving real life — every habit completion triggers satisfying particle bursts, XP flies into the progress bar, and the world itself evolves as you grow.
